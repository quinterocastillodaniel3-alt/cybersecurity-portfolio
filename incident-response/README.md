# Incident Response & Threat Mitigation 🚨

Welcome to the Incident Response section of my portfolio. 

This folder contains reports, post-mortem analyses, and documentation of simulated and real-world security incidents. The projects housed here demonstrate my ability to detect, contain, mitigate, and recover from cyber attacks using industry-standard frameworks like the NIST Cybersecurity Framework (CSF).

### Core Focus Areas:
* Incident Analysis & Triage
* NIST Cybersecurity Framework (CSF) Application
* Threat Containment & Eradication
* Post-Incident Recovery & Documentation

Each subfolder within this directory details a specific security event, outlining the attack vector, the immediate response protocols executed, and the long-term remediation strategies implemented to secure the network.
