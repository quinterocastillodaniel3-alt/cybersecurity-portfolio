Welcome to the **Incident Response** directory of my cybersecurity portfolio. 

This section serves as a centralized hub for technical reports, post-mortem analyses, and documentation of simulated and real-world security incidents. Combining an engineering mindset with advanced security methodologies, the projects housed here demonstrate my ability to systematically detect, contain, mitigate, and recover from cyber attacks using industry-standard frameworks like the NIST Cybersecurity Framework (CSF) and the NIST SP 800-61 Incident Response Life Cycle.

### 🎯 Core Focus Areas:
* **Incident Analysis & Triage:** Rapid evaluation of alerts and IoC extraction.
* **NIST Framework Application:** Structured response utilizing Identify, Protect, Detect, Respond, and Recover phases.
* **Threat Containment & Eradication:** Strategic isolation of compromised assets (endpoints, networks, identities).
* **Post-Incident Recovery:** Lessons learned, root cause analysis, and security control enhancement.
