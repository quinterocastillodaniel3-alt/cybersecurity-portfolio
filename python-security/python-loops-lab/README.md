## 📌 Project Description
In this project, I utilized iterative statements in Python to automate repetitive security measures, such as investigating multiple IP addresses attempting network connections and generating unique employee IDs for a department.

---

## 🔍 Phase 1: Iterative Task Automation
The lab required writing custom logic using both `for` and `while` loops to parse data and execute repetitive commands.

*   **For Loops:** Used the `range()` function and an `i` loop variable to display connection failure messages a specific number of times.
*   **List Iteration:** Wrote a `for` loop to iterate through a list of `ip_addresses` to verify if they existed within an `allow_list`.
*   **Loop Termination:** Implemented the `break` keyword to terminate the loop early if an unapproved IP address attempted to access restricted information.
*   **While Loops:** Created a `while` loop to generate unique employee IDs between `5000` and `5150` that are divisible by 5, incrementing the variable inside the loop body.

## 📝 Phase 2: Key Takeaways
*   Iterative statements are crucial for automating repeated security-related processes.
*   `For` loops allow a process to be repeated a explicitly specified number of times.
*   `While` loops repeat a process continuously until a predefined condition is met.
*   Comparison operators (`<`, `<=`, `==`) are frequently used within the conditions of `while` loops to control execution.
