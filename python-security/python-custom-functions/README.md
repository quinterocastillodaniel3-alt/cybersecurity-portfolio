## 📌 Project Description
In this project, I acted as a security analyst writing Python code to automate repetitive tasks by defining and calling custom functions[cite: 13]. The primary goal was to create reusable blocks of code to display security alerts and format user data[cite: 13].

---

## 🔍 Phase 1: Function Definition & Execution
During this lab, I structured user-defined functions to process information efficiently.

*   **Basic Functions:** Defined an `alert()` function to output a standardized security issue message whenever called[cite: 13].
*   **Loop Integration:** Upgraded the `alert()` function to include a `for` loop, allowing the warning message to display multiple times per function call[cite: 13].
*   **String Concatenation:** Developed a `list_to_string()` function to iterate through a list of approved usernames (`username_list`)[cite: 13].
*   **Data Formatting:** Used the addition operator (`+`) inside the loop to merge individual usernames into a single string variable (`sum_variable`), separated by commas and spaces for readability[cite: 13].

## 📝 Phase 2: Key Takeaways
*   The `def` keyword, followed by the function name, parentheses, and a colon, establishes the function header[cite: 13].
*   The function body consists of an indented block of code that instructs the computer on what actions to execute when called[cite: 13].
*   String concatenation combines multiple strings into one, which is highly useful for formatting list data into readable text files[cite: 13].
