## 📌 Project Description
This project focuses on the fundamentals of programming in Python within a notebook-based coding environment. As a security analyst, writing code helps automate instructions to the computer in various security contexts. 

---

## 🔍 Phase 1: Environment & Syntax Practice
The lab tasks involved interacting with both text cells (markdown) and code cells to execute basic commands.

*   **Markdown Formatting:** Utilized markdown language to format plain text and document the process within the notebook.
*   **Code Comments:** Wrote comments using the hash symbol (`#`) to document the intention behind technical decisions.
*   **Displaying Output:** Used the `print()` function to output specific strings to the screen, such as `"Hello world!"` and `"I am a security analyst."`.

## 📝 Phase 2: Key Takeaways
*   Code comments are ignored by computers but are essential for analysts and teams to understand the intentions behind the code.
*   Comments in Python are initiated using the hash symbol (`#`).
*   When utilizing the `print()` function to display a string, the quotation marks around the string do not appear in the final screen output.
