# 📚 Cybersecurity Study Guides & Theoretical Foundations

Welcome to the **Study Guides** directory. This section serves as a centralized knowledge base compiling foundational and advanced cybersecurity concepts, frameworks, and programming methodologies. 

These guides have been structured to bridge the gap between industrial/embedded systems (IoT/Mechatronics) and enterprise network defense, serving as a robust theoretical preparation for advanced studies in Information Security.

---

## 🗂️ Directory Index

Below is a breakdown of the study modules available in this directory. Click on any folder to access the detailed markdown guide.

### 1. [IAM-Architecture-Guide](./IAM-Architecture-Guide/)
* **Focus:** Identity and Access Management.
* **Topics:** Authentication factors (MFA), Authorization models (RBAC vs. ABAC), Single Sign-On (SSO) architecture, and the AAA framework.

### 2. [Log-Formats-and-Parsing](./Log-Formats-and-Parsing/)
* **Focus:** Log structures and data normalization.
* **Topics:** Standardized parsing formats including Syslog, JSON, XML, CSV, and CEF (Common Event Format).

### 3. [Malware-and-Web-Exploits](./Malware-and-Web-Exploits/)
* **Focus:** Attack vectors and application-layer vulnerabilities.
* **Topics:** Social engineering tactics, malware classification (Ransomware, Trojans), and web exploits like Cross-Site Scripting (XSS) and SQL Injection (SQLi).

### 4. [Network-Traffic-Analysis](./Network-Traffic-Analysis/)
* **Focus:** Network visibility and packet inspection.
* **Topics:** TCP/IP model deep dives, IPv4 header anatomy, and practical usage of CLI protocol analyzers like `tcpdump` and `Scapy` for anomaly detection.

### 5. [Python-Core-Concepts-Guide](./Python-Core-Concepts-Guide/)
* **Focus:** Python programming fundamentals.
* **Topics:** Built-in functions, user-defined functions (`def`), importing external modules, and code documentation standards.

### 6. [Python-Functions-and-Modules](./Python-Functions-and-Modules/)
* **Focus:** Code reusability and organization.
* **Topics:** Function parameters/arguments, the Python Standard Library vs. third-party libraries (e.g., NumPy), and PEP 8 style guidelines.

### 7. [Python-Security-Automation](./Python-Security-Automation/)
* **Focus:** Python for SecOps automation.
* **Topics:** Data types, conditional logic (`if`/`else`), and iterative loops (`for`, `while`) to reduce manual effort in security operations.

### 8. [SIEM-and-Network-Monitoring](./SIEM-and-Network-Monitoring/)
* **Focus:** Log analysis and network visibility.
* **Topics:** Log anatomy, NIDS vs. HIDS, Suricata signature headers, and SIEM data querying (Splunk and Chronicle YARA-L).

### 9. [SecOps-SIEM-SOAR](./SecOps-SIEM-SOAR/)
* **Focus:** Security Operations Center (SOC) infrastructure.
* **Topics:** Intrusion Detection/Prevention Systems (IDS/IPS), log lifecycle, and automated incident response using SOAR playbooks.

### 10. [Vuln-Management-Threat-Modeling](./Vuln-Management-Threat-Modeling/)
* **Focus:** Proactive defense and risk assessment.
* **Topics:** The attacker mindset, CVSS v3.1 scoring, vulnerability scanning methodologies, and the PASTA threat modeling framework.

### 11. [incident-response](./incident-response/)
* **Focus:** Incident handling and enterprise recovery.
* **Topics:** The NIST Incident Response Lifecycle, triage procedures, evidence chain of custody, and conducting effective post-mortem meetings.
